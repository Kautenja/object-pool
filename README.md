# Usage

Simply add [include/object_pool.hpp](include/object_pool.hpp) to
your C++ project either by copying directly or using git submodules.

## Testing

To compile and run the [test suite](test):

```shell
scons test
```

## Benchmarking

To run benchmarks:

```shell
scons benchmark/benchmark.cpp
```
